# Skills Matrix: Full-Stack Engineering & AI Vibe Coding

This document outlines the core competencies, technical stack, and engineering mindset required to execute high-end, AI-driven applications (such as the "Premium Brick" storefront).

## 1. AI Orchestration & Vibe Coding
*   **Prompt Engineering:** Mastering the CO-STAR framework (Context, Objective, Style, Tone, Audience, Response) and advanced reasoning techniques (Tree of Thoughts, Least-to-Most) to guide AI outputs effectively.
*   **AI Tool Mastery:** Selecting the right AI builder for the task—using Lovable for rapid, zero-code full-stack generation, or Cursor IDE for deep codebase manipulation and multi-file refactoring.
*   **Advanced Editor Extensions:** Utilizing Browser MCP to clone and analyze the structural architecture of existing websites, and using the Stage Wise extension for surgical, precise UI tweaks without breaking existing layouts.
*   **The Vibe Coding Pipeline:** Systematically moving from Idea → PRD (Product Requirement Document) → Design Doc → `todo.md` → AI Build → Human Review (via CodeRabbit).

## 2. Frontend Engineering & Design Systems
*   **Core Frameworks:** Building robust interfaces using React, Next.js, Tailwind CSS, and implementing TypeScript with Object-Oriented Programming (OOP) for strict type safety and scalable codebases.
*   **Design System Implementation:** Authoring implementation-ready UI rules using semantic design tokens (typography scale, spacing, color palettes) rather than raw hex values.
*   **Accessibility (a11y):** Ensuring designs meet WCAG 2.2 AA targets, prioritizing keyboard-first interactions, enforcing focus-visible rules, and mapping required component states (default, hover, focus-visible, active, disabled, loading, error).
*   **Browser-Level Understanding:** Mastering the difference between the Document Object Model (DOM) and Browser Object Model (BOM) to improve real-world debugging.

## 3. High-Energy Interactivity & UX Psychology
*   **Frictionless UX:** Anticipating user needs by removing obstacles (e.g., slow load times, cluttered forms) to combat the modern 8.25-second attention span and ensure instant cognitive simplicity.
*   **Advanced Scroll Animations:** Integrating GSAP (`ScrollTrigger` and `ScrollSmoother`) to leverage native scroll mechanics for "buttery smooth," lag-free parallax and text-reveal effects.
*   **3D Integration:** Using Spline to export and seamlessly embed interactive 3D scenes, allowing users to spin and interact with products (like "The Monolith" brick) directly in the browser.
*   **Marketing Psychology:** Applying luxury marketing tactics (inspired by "The Markup Marché") to artificially inflate perceived value through exaggerated, scientific-sounding language.

## 4. Backend Architecture & Security
*   **Modular Pattern Design:** Moving beyond single-file logic to separate Controllers, Services, Routes, and Models, ensuring enterprise-level maintainability in a team environment.
*   **Database Engineering:** Utilizing MongoDB and enforcing strict data validation through Mongoose ODM schemas to prevent data inconsistency.
*   **Authentication & Payments:** Implementing Role-Based Access Control (RBAC) and integrating payment gateways like Stripe and SSLCommerz for complete product logic.

## 5. Engineering Mindset & Version Control
*   **Git & GitHub Mastery:** Utilizing command-line Git for version history (`git init`, `git commit`, `git checkout`), safely branching logic, managing the staging area, and protecting sensitive data with `.gitignore` files.
*   **System Thinking:** Moving beyond mere syntax memorization to focus on problem breakdown, system architecture, and understanding client-server data flows.
*   **Automated Quality Assurance:** Integrating CodeRabbit into pull requests to automatically scan for bugs, enforce coding guidelines, and catch edge cases humans might miss before code goes to production.
