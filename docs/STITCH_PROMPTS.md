# 🪄 Stitch Prompts

A record of the [Google Stitch](https://stitch.withgoogle.com/) prompts used to generate the visual design for Brickhunter, organized by section and in the order they were written.

## How to Use This File

- Each prompt is shown **verbatim** in a code block, exactly as entered into Stitch — do not paraphrase when adding new entries.
- Below each prompt is a short note on what Stitch generated in response, and how (or whether) it made it into the final build.
- Screenshot placeholders sit below each entry — replace them with real exports from Stitch or the running app before publishing.
- When revising a section, add a new versioned entry (e.g. "Hero Section (v2)") rather than overwriting the original — this file is a history, not just a snapshot of the current design.
- If a prompt was iterated on conversationally (follow-up refinements within the same Stitch session), include the follow-up as a separate labeled sub-prompt under the same section.
